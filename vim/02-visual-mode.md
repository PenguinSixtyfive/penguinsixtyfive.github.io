# Visual mode

Visual mode is used to select text.  
After the text is selected, you can manipulate it with the shortcuts below.

| Key combo | Command |
| --- | --- |
| v | Visual mode |
| V | Visual line mode |

### Cut, copy & paste:
| Key combo | Command |
| --- | --- |
| d | Cut (delete) |
| y | Copy (yank) |
| p | Paste (put) |
