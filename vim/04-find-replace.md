# Find & replace

Find and replace words and sentences.

| Key combo | Command |
| --- | --- |
| /<search term> | Find term |
| :%s/<search term>/<replace with> | Replace old text with new |
| :%s/\<search\>/\<replace\>/gc  | where `g` = global (all document), and `c` = confirm |
